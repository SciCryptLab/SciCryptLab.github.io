Your whole pipeline as one interactive graph.

What it is

A live, navigable map of your dataset. Nodes are files and script steps; edges are the actual data dependencies. The graph is derived from your commit history plus a local LLM analysis of each script's reads and writes — so it reflects what really happened, not a diagram you had to draw.

What it can do

  • Click to explore. Select any node to preview its contents, see its metadata, or open the script that wrote it.
  • Run from the graph. Execute a script directly from its node and watch the outputs appear downstream.
  • Colour-coded by type. Raw data, scripts, results and plots are instantly distinguishable.
  • Risk tinting. Steps flagged by manipulation detection light up orange or red, right where they happen.
  • Stage-level operations. Save, inspect, or restore from any point without leaving the view.
