Everything SciTrace does.

A native Git + git-annex engine under a web interface — no command line. Pick a capability to go deeper.

Native data versioning

Structured dataflows (raw_data/ scripts/ results/ plots/) versioned with Git + git-annex. Restore any file from any point in history — from the browser.

Learn more →

Interactive dataflow editor

A Vis.js graph of files and scripts. Click any node to preview, run, or inspect. Colour-coded by type, with lineage derived automatically.

Learn more →

Provenance run records

Run a script through SciTrace and the command, inputs, outputs and exit status are written into a [SciTrace:Run] commit. Provenance, free.

Learn more →

Hardware-secured access

Passwordless WebAuthn/FIDO2 login with YubiKey, Touch ID or Windows Hello, plus on-device decryption and revocable API tokens.

Learn more →

Projects & task tracking

Group dataflows into projects with collaborators, and track the work with per-project tasks, priorities and statuses.

Learn more →
