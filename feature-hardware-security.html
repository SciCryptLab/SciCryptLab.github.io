SciTrace · feature

Passwordless by design, hardware-backed where it counts.

Sign in with a security key, a fingerprint, or Windows Hello — no passwords to phish. Sensitive actions step up to a fresh hardware verification, and encrypted payloads can be decrypted on-device by a YubiKey that never reveals its secret.

What it is

SciTrace supports WebAuthn / FIDO2 passwordless authentication and integrates hardware keys for the operations that matter most. For production decryption, a PKCS#11 integration with YubiKey 5-series performs the unwrap on the device itself — PIN and touch required — so the key material never leaves the token.

What it can do

  • Passwordless login. YubiKey, Touch ID, or Windows Hello — nothing to remember, nothing to phish.
  • Step-up verification. Sensitive actions demand a fresh hardware confirmation.
  • On-device decryption. PKCS#11 + YubiKey unwraps encrypted payloads without exposing the key.
  • Scriptable, revocable tokens. Bearer API tokens (sct_*) are stored only as SHA-256 hashes and can be revoked at any time.
