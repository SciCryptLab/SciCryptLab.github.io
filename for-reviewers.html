The problem with the system today

You get the PDF and a cover letter. The data is somewhere else.

Methods sections omit the load-bearing step. Supporting data hides in an SI document, behind a paywall, or in a Dropbox link that will 404 in eighteen months. By the time you can ask, the author has moved on. And the unstated assumption is that you'll spot any data-manipulation issue from text alone.

How SciCrypt solves it

Three tabs. The manuscript, the lineage, your decision.

  • Assignment tab. Abstract, manuscript files, your accept/decline status.
  • Data Lifecycle tab. The full SciTrace Vis.js graph the author linked — every input, every script, every output. You can see for yourself whether Fig. 3 really did come from filter_and_clean_samples.py running on the published raw data.
  • Decision tab. Recommendation form + a private comment thread with the editor that the author never sees.
